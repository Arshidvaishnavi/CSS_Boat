# Animated Drawing Project

This project is a **drawing and animation** made using **HTML and CSS**.  
It includes multiple shapes, lines, and animations to create a creative visual effect.

---

## Features

- Multiple shapes and lines using `div` elements  
- Animated water effect using CSS keyframes  
- Shapes arranged to create a visual drawing  
- Simple CSS animations for movement  

---

## Technologies Used

- HTML  
- CSS  

> The project demonstrates creativity with **pure HTML and CSS** without any JavaScript or frameworks.

---

## Live Demo

Check out the live demo here: https://arshidvaishnavi.github.io/CSS_Boat/

## How to View

1. Download the repository from GitHub  
2. Open the file `index.html` in any web browser  
3. You will see the animated drawing and water animation
